# Air Force Duct Cleaning & Testing — Website

A fully static, multi-page website for [Air Force Duct Cleaning & Testing](https://feeltheairforce.com/) based in Conway, AR.

## Pages
- `index.html` — Home
- `services.html` — Services
- `faq.html` — FAQ
- `reviews.html` — Reviews
- `contact.html` — Contact
- `signin.html` — Sign In (localStorage auth)
- `register.html` — Create Account (localStorage auth)

## Hosting on GitHub Pages
1. Create a new GitHub repository
2. Upload this entire folder's contents to the repo root
3. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Site will be live at `https://yourusername.github.io/your-repo-name/`

No build step. No dependencies. Pure HTML/CSS/JS.
